Notice: KnarkOS is a work in progress! It is currently being developed as a hobby project.

Imagine an operating system where Emacs takes center stage, serving as the foundation and written entirely in a modern programming language. The goal of KnarkOS is to become a complete operating system that uses Lisp as its core and builds everything on top, like Emacs.
